# Universal Sizing Study
A set of Python code modules that explores the stages needed to construct a hydrogen atom described in the Natural Philosophy of Universal Matter.
From VS Code: Open the file main.py and execute it running with or without debug mode.
From the command line: Open a terminal (or commanbd window) and type python3 main.py. 

In either case, the output (report) will found in the text file sizing_output.txt.
